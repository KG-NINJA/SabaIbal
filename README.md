# SabaIbal

HTMLだけで構成した2Dクラフトサバイバルゲームです。昼は素材を集めて建築し、夜に出没する怪物から身を守りましょう。

## 遊び方

1. `index.html` をブラウザで開きます。
2. プレイヤーは WASD か矢印キーで移動できます。
3. スペースキーで近くの資源を採取したり、動物や怪物を攻撃できます。
4. Bキーで正面のマスに木材2個を消費して壁を建築できます。
5. 昼の間に木材・石材・肉を集めて装備を整え、夜の怪物に備えましょう。

## ゲームの特徴


- 広い 30×20 マスのフィールドで探索と建築が可能
- 昼夜サイクル (昼と夜はそれぞれ約60秒)
- 昼は動物が現れ、狩りで肉を確保できます
- 夜は怪物がスポーンし、ゆっくりとプレイヤーを追跡します

- 木材を消費して防壁を建設し、基地を防衛可能
- シンプルなHUDで体力・資源・日数を表示

怪物に体力を奪われて0になるとゲームオーバーです。生き延びられる日数に挑戦してみてください！
